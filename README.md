# Tracko 🚀

A modern productivity tracker with:
- Task management
- XP & Level system
- Badges
- Dashboard UI

Built using HTML, CSS, JavaScript.

## 🔥 Features

- Add and manage tasks
- Track progress with XP system
- Level up based on productivity
- Unlock badges
- Modern dashboard UI

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

## 🚀 Future Improvements

- Backend (Node.js + Database)
- User authentication
- Cloud sync
